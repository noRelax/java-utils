Readme for iChat LE 1.1

This version is ONLY for Windows users.

Extract this zip file to any path. Inorder to have reminding sound when new message comes, please copy the two .jar file to your $JRE_HOME$\lib\ext directory, where $JRE_HOME$ is the directory of Java runtime enviroment.

This should work for most users. Contact allenchue@gmail.com when having problems.
